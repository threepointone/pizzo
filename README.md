# Think Basic Starter

A minimal [Think](https://www.npmjs.com/package/@cloudflare/think) chat agent
running on Cloudflare Workers, with a small React chat UI.

Out of the box you get streaming responses, persistent message history,
resumable streams, and built-in workspace file tools.

## Develop

```sh
npm install
npm run dev
```

Open the printed URL and start chatting.

## What's here

- `agents/assistant/agent.ts` — the agent. Override `getModel` and
  `getSystemPrompt`, add `getTools()`, `configureSession()`, skills, and more.
- `src/client.tsx` — a React chat client using `useAgent` + `useAgentChat`.
- `wrangler.jsonc` / `think.d.ts` — generated by the Think framework. Re-run
  `npm run types` after changing agents or bindings.

## Deploy

```sh
npm run deploy
```
